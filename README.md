## Overview

This project demonstrates how to build a RESTful API using .NET 10 and connect it to a cloud-hosted relational database using Azure SQL Database.

This project illustrates how modern backend systems integrate .net api with managed cloud databases.

---

## Architecture

```
Client
   ↓
ASP.NET Core Web API
   ↓
Entity Framework Core
   ↓
Azure SQL Database
```

## Technologies Used

* .NET 10
* Entity Framework Core
* Azure SQL Database
